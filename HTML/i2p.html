I2P Server Configuration

Where possible I have linked and pre-configured Monero settings to work with extrnal services such as this. I2P however requires your manual configuration.

To install I2P first head over to the Web Terminal and select "Install I2P Server" from option (4) of the "Extra Network Tools" menu. Installation takes less than 5 minutes.

Once installed the button bellow will take you to the I2P menu. For how to configure your I2P Server you can refer to my usage demo
